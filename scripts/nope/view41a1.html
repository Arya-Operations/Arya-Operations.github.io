Macros

This script allows you to create keybinds for client commands. Then you can simply press a key on your keyboard to execute a command instead of typing it out each time.

Readme

This script allows you to create keybinds for client commands. Then you can simply press a key on your keyboard to execute a command instead of typing it out each time.

This script is open source! Check it out on GitHub.

Usage

Make sure the module called Macros is enabled.

Adding/removing macros

.macros add ˂key˃ ˂command˃
.macros remove ˂key˃
Example
.macros add z say /home
.macros remove z

Execute a macro

Simply press the key a macro is bound to to execute it.

Example usages

  • To quickly execute commands like /home.
  • To change modes of modules.
  • To send a message in the chat.
